# Dashboard comercial · Instituto Bruner

Panel mensual para asesoras y supervisión de Instituto Bruner.

## Funciones

- Acceso individual por asesora.
- Objetivo personal y ritmo diario sugerido.
- Objetivo grupal administrado por supervisión.
- Ventas y detalle de inscripciones desde las planillas institucionales.
- Comisión estimada y simulador independiente.
- Vista supervisora con ranking, objetivos y cuadro de comisiones.

El backend y las credenciales no forman parte de este repositorio público.
